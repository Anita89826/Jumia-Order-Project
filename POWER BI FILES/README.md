#  Jumia Order Data Analysis – Power BI Project

##  Project Overview  
This project analyzes 3,000 Jumia customer orders to uncover insights about regional performance, product categories, and customer behavior. The analysis was conducted using Power BI for data cleaning, modeling, and interactive dashboard creation.

---

##  Key Insights  
- **Top Regions:** East and South lead in total revenue.  
- **Best-Selling Categories:** Food & Beverages dominate, followed by Electronics.  
- **High-Value Customers:** Doctors and Students make the highest average purchases.  
- **Income vs Quantity:** Mid-income buyers (₦40K–₦80K) purchase more items per order, while high-income customers make fewer but premium purchases.  
- **Gender Distribution:** Balanced revenue between male and female customers.

##  Business Recommendations  
- Focus marketing on **East and South** regions.  
- Create **bundle offers** for mid-income, high-volume customers.  
- Develop **premium campaigns** for higher-income customers.  
- Keep **gender-neutral** promotions given balanced spending patterns.
